The Spark File

17th March 2021 at 3:46pm

Store every failed poem, writing line and idea in one big file to sort through later. Comb through it regularly when unispired and see what jumps out at you.

Ideas compost, the spark file is the compost pile where things break down and combine in new ways.

I've kept a spark file for years and it's massive. Because it's plain text, it's still a compact file and able to be easily searched and looked through. I have it stored as an org file in emacs and have a function to show a random headline.

It's good to look through years later and see the start of ideas.

  • This idea comes from [Stephen Johnson](https://medium.com/the-writers-room/the-spark-file-8d6e7df7ae58)
