Starting a poem

Here's some ways I've found to start a poem.

  • Use a prompt
  • Free Write to a timer
  • Write an ekphrastic poem
  • Writing Exercises
  • Embrace randomness
  • How are you feeling? Write around that

Then look at Ways to edit a poem

It's important to try to focus.
